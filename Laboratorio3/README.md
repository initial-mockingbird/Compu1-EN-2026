# Instrucciones para el Quiz

- El quiz posee 4 partes, cada una con un valor de 20 puntos, para un total de 100 puntos.
- El valor de cada pregunta dentro de cada parta es igual, es decir, si la parte 1 tiene 5 preguntas, entonces cada pregunta vale 20/5 puntos.
- Cada parte contiene un archivo `README.MD` con las instrucciones para esa parte.
- Finalmente, el quiz solamente se evaluara si responde la encuesta de final de curso [proveida aqui](https://forms.gle/iSVioWrRbYzM4xvU7)
